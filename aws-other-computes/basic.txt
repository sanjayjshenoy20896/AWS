Docker:
It is a software development platform to deploy apps
Apps are packaged in containers that can be run on any OS
Apps runs same , regardless of where they are run
1.) Any Machine
2.) No compatibility issues
3.) Predicatble behavior
4.) less work
5.) Easier to maintain and deploy
6.) Works with any language,any OS, any tech

Scale containers up and down very quickly.

Multiple docker containers on same ec2 instance

Where Docker images are stored?
They are stored in docker repositories

1.) Public: Docker Hub
find base images for any technology or OS
a.) Ubuntu
b.) MySQL
c.) NodeJS,JAVA
2.)Private: Amazon ECR -> Elastic container registry

Docker vs Virtual Machines
Docker is a sort of a virtualization technology but not exactly

Resources are shared with host => many containers on a single host
