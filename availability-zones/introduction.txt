AWS Shared Responsibility model

The customer is Responsibile for the security "IN" the cloud.
Customer data
Platform, applications,identity and access management
OS, network amd firewall configuration
Client-side data encryption & Data Integrity Authentication
Server-side encryption
Network traffic protection 

AWS is Responsibile for security "OF" the cloud.
Software
Compute
Storage 
Database
Networking
Hardware
Regions,availablity zones and edge Locations


Customer Responsibility: (Security in the cloud)
a.) Customer data
b.) Client-side data encryption

Customer / AWS Responsibility: (Varied Responsibility by service)
a.) Server side encryption
b.) Network traffic protection
c.) Platform and application management
d.) OS, Network and firewall

AWS Responsibility: (Security of the Cloud)
a.) Software for compute storage, database and networking
b.) Hardware and global infrastructure
